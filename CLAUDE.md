# Project Rules

This is a WooCommerce plugin with critical checkout flows.

Use OpenSpec for:
- checkout changes
- REST API changes
- coupon/session logic
- order submission
- payment flow
- external API integrations
- deploy flow modifications

Before implementation:
1. Create proposal
2. Define requirements
3. Define edge cases
4. Generate tasks
5. Then implement

Follow:
- WordPress Coding Standards
- WooCommerce hook best practices
- Existing namespace structure
